Ayuda rapida

Guia funcional de pantallas, campos y efectos del firmware. Esta pagina esta pensada para consulta rapida desde el propio dispositivo.

Pantallas Campos clave Efectos Flujos rapidos

Pantallas principales

Home

Pantalla principal para colores, paletas, efecto activo, parametros, brillo, lista de secuencia y estado runtime.

Configuracion

Indice de acceso a Network, Microfono, GPIO, Profiles, Paletas, General y Manual JSON.

Network

Editor completo de WiFi, hostname, NTP y direccionamiento AP/STA.

Microfono

Ajuste fino del micro I2S: ganancia, floor, gate, AGC y pines.

GPIO

Definicion de salidas LED y limite de consumo estimado.

Profiles

Snapshots completos del dispositivo para guardar, clonar, editar y activar.

Paletas

Catalogo del sistema y editor inline de paletas de usuario.

General

Idioma de la interfaz, region, debug y heartbeat del sistema.

Manual JSON

Importacion/exportacion completa de configuracion con validacion previa.

API

Entradas de prueba para los endpoints REST principales.

Version

Resumen de firmware, hardware, memoria, watchdog y reinicio de la placa.

Flujos rapidos recomendados

Cambiar un efecto

Home -> elegir efecto -> ajustar secciones, rapidez, nivel y brillo -> Aplicar.

Dejar un arranque fijo

Home -> preparar estado deseado -> Guardar arranque.

Crear un perfil completo

Config -> Profiles -> Guardar configuracion actual como perfil.

Ajustar sensibilidad de audio

Config -> Microfono -> ajustar gain, noise floor, noise gate y AGC response mientras un efecto de audio esta activo.
